# SanWich v2.5：本地私密 AI 與低延遲字幕播放器

> 開發版：**v2.5b**  
> 正式發佈目標：**v2.5**  
> 核心目標：音訊與字幕校對文字都能留在使用者電腦，並讓舊電腦上的 SRT 同步預覽可實際使用。

## 一、版本範圍

### 本地私密 AI

- 新增 `local` 供應商，不需要 API Key。
- 採用 Breeze-7B-Instruct v1.0 Q4_K_M，針對台灣繁體中文，模型約 4.54GB。
- 使用 llama.cpp 的 OpenAI 相容本機伺服器，只監聽 `127.0.0.1`。
- llama.cpp 從官方 GitHub Release 下載，NVIDIA GPU 依世代選擇 CUDA 13／CUDA 12，無 NVIDIA GPU 時使用 CPU 資產。
- 所有 runtime 資產與 GGUF 模型都驗證檔案大小及 SHA256。
- 新版 `llama-server.exe` 本身約 9KB、主要功能位於 `llama-server-impl.dll`；完整性檢查會一起驗證 server、impl、llama、ggml 與 CUDA DLL，不能只靠 EXE 大小判斷。
- RTX 50 系列使用 CUDA 13；較舊 NVIDIA 顯卡使用 CUDA 12。
- 依 VRAM 自動限制 context 與 GPU layers；啟動失敗時依序關閉 Flash Attention及退回 CPU。
- 連接埠從 8080 起動態尋找，只綁定 loopback。

### SRT 低延遲預覽

- FFmpeg 保留為格式解碼器，只在載入媒體時建立一次 PCM WAV 代理檔。
- `ffplay` 不再是主要互動式播放器；主路徑改為常駐 `sounddevice`／PortAudio 音訊串流。
- 播放游標改由 DAC 音訊時鐘換算，不再從外部程序建立時間開始猜測。
- 播放、暫停、seek 與 scrub 共用同一條音訊串流。
- WAV 代理檔依來源路徑、大小與修改時間快取於 `%APPDATA%\SanWich\cache\audio_preview`。
- 長音訊波形改為逐桶讀取，不再把完整 PCM 一次載入記憶體。
- 快取上限 8GB，超過 45 天或超過容量時自動清理舊檔。
- 若 PortAudio 無法開啟，保留 `ffplay` 作為相容性備援並寫入日誌。

### 安裝與啟動

- `01_setup.bat` 安裝 `sounddevice`。
- 桌面捷徑優先透過 `run_hidden.vbs` 隱形啟動。
- 主程式輸出仍寫入 `logs/main.log` 與 `logs/main_error.log`，隱形啟動不隱藏診斷資料。
- 新增本地 AI 下載／檢查入口；下載支援 `.part` 續傳。
- 首次下載前檢查 `%LOCALAPPDATA%\SanWich\local_ai` 所在磁碟；7GB 是最低門檻，建議保留 10GB。

### 低顯存處理順序

- 選擇本機私密 AI 時，先停止先前殘留的 `llama-server`，避免它在 Breeze-ASR 啟動前占用顯存。
- 批次工作先完成所有檔案的 ASR；需要語者分離時，也在這一階段完成音訊分析，只保留時間區段資料。
- ASR 階段結束後，將 pipeline 設為 `None`，執行 Python 記憶體回收及 `torch.cuda.empty_cache()`，再啟動本機 LLM。
- 本機 LLM 若無法啟動，仍會輸出 Breeze 原始字幕，並把該檔案列為 AI 校對未完成。

## 二、隱私邊界

- Breeze-ASR-25 語音辨識：本機。
- 選擇「本機私密 AI」時的字幕校對：本機 `127.0.0.1`。
- 音訊、字幕、Prompt、個人化規則不送往第三方 LLM。
- 第一次安裝仍需連網下載 Python 套件、ASR 模型、llama.cpp 與 GGUF 模型；下載請求不包含使用者的媒體或字幕內容。
- 使用者若主動切換 Gemini、OpenAI、Claude 或 DeepSeek，字幕文字仍會依選擇送到該雲端供應商。

## 三、v2.5 正式版驗收門檻

- 2060／6GB VRAM 可啟動本地模型；不足時能清楚退回較少 GPU layers 或 CPU。
- RTX 50 系列不會誤裝 CUDA 12.4 runtime。
- 模型與 runtime 中斷下載後可續傳，且錯誤 SHA256 不會被啟動。
- 本地模式不填 API Key 仍可完整走完 AI 校對與語意斷句。
- SRT 播放器不會在每次播放或拖曳時建立新的 `ffplay.exe`。
- 游標位置取自音訊裝置時鐘，長時間播放不累積漂移。
- 既有字幕、授權、個人規則與學習閉環測試全部通過。
- 分享 ZIP 必須包含 `core/audio_preview.py`、`core/local_llm.py` 與隱形啟動器，且不包含模型、runtime、API Key 或使用者資料。
